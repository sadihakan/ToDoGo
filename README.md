# ToDoGo

Simple Todo Api (using chi and sqlx)

## Getting Started

### Requirements

* Go >= 1.11.x
* PostgreSQL

### Installing

* How to install packages

```
go mod tidy
```

* Create your .env file

```
DB_HOST=127.0.0.1
DB_PORT=9200
DB_NAME=Legendary
DB_USER=TheSpecial
DB_PASS=EverythingIsAwesome
```


### Executing program

* How to run the program

```
go run ./cmd
```
